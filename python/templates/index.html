API endpoints:

Valid User:

Checks to see if the specified user exists.

Returns: True or False

Example: /validuser/{{ data.user.sAMAccountName }}

/validuser/:auid

User Info:

Looks up a user by their auid.

Returns: JSON object

Example: /userinfo/{{ data.user.sAMAccountName }}

/userinfo/:auid

User Groups:

For the specified auid, retrieves all of the groups that the user belongs to.

Returns: JSON array

Example: /usergroups/{{ data.user.sAMAccountName }}

/usergroups/:auid

All Info:

Returns the combined data of the validuser, userinfo, and usergroups endpoints.

Example: /all/{{ data.user.sAMAccountName }}

/all/:auid
